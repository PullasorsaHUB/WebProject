## Frontend käynnistysohjeet

### Asennus

Frontend-kansion juuressa, aja:

```bash
npm install
```

### Kehityspalvelin

Käynnistä kehityspalvelin:

```bash
npm run dev
```

Sovellus on osoitteessa `http://localhost:5173`.

## Prisma-komennot

Kehittäjätyökalut Prisma-tietokantahallintaa varten:

```bash
# Seed-skriptin ajaminen
npx tsx prisma/script.ts

# Prisma Studio avaaminen
npx prisma studio

# Prisma Client uudelleengenerointi
npx prisma generate
```
